# AI-Powered Resume Screening Tool

## Description
This tool automates the process of parsing and ranking resumes based on a job description using Natural Language Processing (NLP). It uses TF-IDF and cosine similarity to score and sort resumes by relevance.

## Features
- Parse resumes (PDF)
- Upload job descriptions
- Rank candidates using cosine similarity
- Export ranked list as CSV

## Tech Stack
- Python, spaCy, scikit-learn, Streamlit, pandas, PyPDF2

## How to Run
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`

## Deployment
You can deploy it on Streamlit Cloud for free.
